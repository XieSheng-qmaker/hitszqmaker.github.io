## 推文同步方法
1. 新建issue，标题随意（一般都写推文主标题）
2. 使用Markown语法，添加下列信息：
```html
<div class="feature is-revealing">
    <div class="feature-inner">
        <div class="feature-icon">
            <img src="推文图片链接">
        </div>
        <h4 class="feature-title h3-mobile">主标题</h4>
        <p class="text-sm">副标题</p>
        <p class="hero-cta is-revealing" style="visibility: visible;">
            <a class="button button-secondary button-shadow" href="推文原文链接">查看更多</a>
        </p>
    </div>
</div>
```
